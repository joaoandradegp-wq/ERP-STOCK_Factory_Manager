Sistema Integrado de Gestão desenvolvido para a empresa R.A Estofados, visando o controle e centralização de todas as atividades exercidas pela mesma:

- Folha de Pagamento.
- Controle de Compras/Vendas/Estoque/Confecções/Matéria-Prima
- Controle de Funcionários/Clientes.
- Fluxo de Caixa.
- Relatórios Gerenciais.

Baseado no levantamento e estudo do fluxo e processos de uma fábrica de móveis e estofados.
